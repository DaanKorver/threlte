Trait components are mostly used internally and make certain aspects of three.js objects
accessible and reactive and interact with Threlte systems. They serve advanced use cases and are
not required for basic usage.

- [`<HierarchicalObject>`](/docs/reference/core/hierarchical-object)
- [`<SceneGraphObject>`](/docs/reference/core/scene-graph-object)
